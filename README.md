## btctrading

Time Series Forecast with Bitcoin value, to detect upward/down trends.

It's presented how Classification Problem multiclass [UP, KEEP, DOWN]

## Install and run:

- pip install -r requirements.txt
- jupyter notebook
- Execute Ensembles.ipynb or EDA.ipynb
